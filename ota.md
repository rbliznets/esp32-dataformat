# Команды для работы с обновлением прошивки
В корне json должен быть только один элемент __"ota"__. Корень json может содержать другие элементы. 
Одновременно передаётся только одна команда, следующая только после получения ответа.
При возникновении ошибки при обработке команды выдаётся следующий ответ:
```
{
    "ota":
    {
        "error":"описание ошибки"
    }
}
```
### 1.Записать данные во флэш.
```
{
    "ota":
    {
        "data":"е90a2020202022756470223a207b0a20202020202020202273736964223a20225265646d695f39343330222c0a20202020202020202270617373776f7264223a2022466f7874726f7431222c0a2020202020202020226465" //данные
        "mode":"end"      // end - завершение записи данных в флэш, begin - отмена предыдущей записи и начало новой
    }
}
```
Ответ
```
{
    "ota":
    {
        "offset":88,         //всего записано байт
        "mode":"end"        //если запись прошла успешно
    }
}
```

Первые данные должны начинаться с magic word (0xE9), а последние заканчиваться контрольной суммой.
