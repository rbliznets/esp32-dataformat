# Команды для работы с nvs
В корне json должен быть только один элемент __"nvs"__. Корень json может содержать другие элементы. 
Одновременно передаётся только одна команда, следующая только после получения ответа.

### 1.Очистить nvs.
```
{
    "nvs":
    {
        "clear":null
    }
}
```
Ответ
```
{
    "nvs":null
}
```
### 2.Прочитать данные.
```
{
    "nvs":
    {
        "name":"rssi",  //имя поля
        "type":"u16"    //тип поля (u8, u16, u32, i8, i16, i32)
    }
}
```
Ответ
```
{
    "nvs":
    {
        "name":"rssi",  //имя поля
        "value":300     //значение поля, если поле не существует, то оно отсутсвует
    }
}
```
### 3.Записать данные.
```
{
    "nvs":
    {
        "name":"rssi",  //имя поля
        "type":"u16",    //тип поля (u8, u16, u32, i8, i16, i32)
        "value":300     //значение поля
    }
}
```
Ответ
```
{
    "nvs":
    {
        "name":"rssi",  //имя поля
        "value":300     //значение поля
    }
}
```
### 4.Перегрузить устройство.
```
{
    "nvs":
    {
        "reset":null  
    }
}
```
