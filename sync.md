# Команды для работы с синхронизацией времени
В корне json должен быть только один элемент __"sync"__. Корень json может содержать другие элементы. 
Одновременно передаётся только одна команда, следующая только после получения ответа.
При возникновении ошибки при обработке команды выдаётся следующий ответ:
```
{
    "sync":
    {
        "error":"описание ошибки"
    }
}
```

### 1.Установить текущее время.
```
{
    "sync":
    {
        "epoch":1725192000, // время в секундах от 1 января 1970 года. Если поле отсутствует, то должно быть "force".
        "force":true // если true, то время будет установлено даже если уже была синхронизация. Если поле отсутствует, то false
    }
}
```
Ответ
```
{
    "sync":
    {
        "result":true, // true - установлено и записано, false - не установлено, но время уже синхронизировано
        "epoch":1725192000 // текущее время устройства в секундах от 1 января 1970 года.
    }
}
```
